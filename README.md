# Quantum Vault: Secure Token Authentication Without Classical Side Information Benchmarked on IBMQ

This repository contains the basic methods for implementing the quantum vault protocol with IBMQ from 'Quantum Vault: Secure Token Authentication Without Classical Side Information Benchmarked on IBMQ' by L. Tsunaki et al., available at: [arXiv:](https://arxiv.org/abs/). 

A script for implementing the SWAP test can be found at [swap_test.py]().
The IBMQ calibration reports from the day of the implementations 08.04.2026 can be found at [calibration_reports]().

This application has dependencies on NumPy version 2.3.1, Qiskit version 2.1.0 and Qiskit IBM Runtime version 0.45.0.
Due to Qiskit's fast development, the code is not guaranteed to work with newer versions as the ones specified above.
